## a)

To determine ephemeris validity, we should check the `fit interval flag` in `IS_GPS_200J`. On page 45, we see that there are three Operational Intervals: Normal Operations, Short-term Extended Operations, and Long-term Extended Operations. If SV is operating normally, the fit interval flag is zero. Unless the fit interval flag was 1, to find out the type of operation interval, we should additionally look for IODE. If  IODE is less than 240, then SV is undergoing short-term extended operations, and if it is in the range 240-255, then SV is undergoing long-term extended operations.

On page 131, the corresponding curve fit interval in Normal operations is four hours. TOE corresponds to the center of the curve fit, that's why the validity of this ephemeris data is +/-2 hours of TOE.

```matlab
p15_sv = SV(15);
p15_dataset = p15_sv.navData;

p15_toe = p15_dataset.TOE;


function [dayOfWeek, time] = inferTime(dt)
    dayOfWeek = floor(dt / 86400);
```

```matlab
function ek = calc_ek(e, mk)
    ek = mk;
    dE = ek;
    while(abs(dE) > 1e-12)
        E_old = ek;
        ek = mk + e * sin(ek);
        dE = ek - E_old;
    end
end
```

```matlab
function plot_orbit(SV, prn, t0, freez)
    a = SV(prn).navData.sqrtA ^ 2;
    mu_earth = 3.986005 * (10^14);
    T = 2 * pi * sqrt(a^3 / mu_earth);
    revolutions = ceil(T);
    r_ECEF = zeros(3, revolutions);
    lat    = zeros(1, revolutions);
    lon    = zeros(1, revolutions);
    
    for k = 0:revolutions
        if freez
            SV = svPosVelECEF(SV, prn, t0 + k, 4, 0);
        else
            SV = svPosVelECEF(SV, prn, t0 + k, 4);
        end
        r_ECEF(:, k+1) = SV(prn).POSITION(4, :);
        lon(k+1) = rad2deg(atan2(r_ECEF(2,k+1), r_ECEF(1,k+1)));
        r_mag = norm(r_ECEF(:, k+1));
        lat(k+1) = rad2deg(asin(r_ECEF(3,k+1) / r_mag));
    end
    
    figure('Name', '3D ECEF Orbit Simulation', 'Color', 'w', 'Position', [100 100 800 600]);
    hold on; grid on; axis equal;
    
    % Load built-in MATLAB topographic data for a realistic Earth
    load topo; 
    [X, Y, Z] = sphere(50);
    
    R_earth = 6371000;
    
    % Plot the Earth with Topography
    surf(X*R_earth, Y*R_earth, Z*R_earth, 'CData', topo, ...
        'FaceColor', 'texturemap', 'EdgeColor', 'none');
    
    % Plot the ECEF Orbit Trajectory
    plot3(r_ECEF(1, :), r_ECEF(2, :), r_ECEF(3, :), 'r', 'LineWidth', 1.5);
    
    % Plot Start and End points
    plot3(r_ECEF(1, 1), r_ECEF(2, 1), r_ECEF(3, 1), 'go', 'MarkerFaceColor', 'g', 'MarkerSize', 6);
    plot3(r_ECEF(1, end), r_ECEF(2, end), r_ECEF(3, end), 'ko', 'MarkerFaceColor', 'y', 'MarkerSize', 6);
    
    title('Satellite Orbit in ECEF Frame (Earth is Fixed)');
    xlabel('X (m)'); ylabel('Y (m)'); zlabel('Z (m)');
    view(3);
    legend('Earth Surface', 'Trajectory (Shifts Westward)', 'Start', 'End');
    hold off;
    
    %% Visualization 2: 2D Ground Track
    % Fix the line wrap-around issue in plotting from +180 to -180 longitude
    for i = 1:length(lon)-1
        if abs(lon(i+1) - lon(i)) > 180
            lon(i) = NaN; % Insert a break in the plot line
        end
    end
    
    figure('Name', '2D Ground Track', 'Color', 'w', 'Position',[950 100 800 400]);
    hold on; grid on;
    
    % Plot a flat 2D map of the Earth
    contour(0:359, -89:90, topo, [0 0], 'k'); % Draws coastlines using topo data
    axis([0 360 -90 90]);
    
    % Shift longitude from [-180, 180] to [0, 360] for easier map plotting
    lon_map = mod(lon + 360, 360);
    
    % Plot the ground track
    plot(lon_map, lat, 'r.', 'MarkerSize', 5);
    plot(lon_map(1), lat(1), 'go', 'MarkerFaceColor', 'g', 'MarkerSize', 8);
    plot(lon_map(end), lat(end), 'ko', 'MarkerFaceColor', 'y', 'MarkerSize', 8);
    
    title('Satellite Ground Track');
    xlabel('Longitude (Degrees)');
    ylabel('Latitude (Degrees)');
    legend('Coastlines', 'Ground Track', 'Start', 'End', 'Location', 'northeast');
    hold off;
end
```
